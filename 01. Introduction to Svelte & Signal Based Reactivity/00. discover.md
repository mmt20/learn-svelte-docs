# Section 1 : Svelte Fundamentals & Reactivity

## Core Reactivity
- **Signal-Based Model**: Familiarized with the fine-grained reactivity system in Svelte 5.
- **Runes**: Deep dive into `$state`, `$derived`, `$derived.by`, `$effect`, and `$effect.pre`.
- **Pure Functions**: Understood why derivations and components should remain pure.
- **Microtask Queue**: Learned how Svelte schedules effects asynchronously for performance.
- **Untrack**: Using `untrack()` to read state inside an effect without creating a dependency.
- **Tick**: Using `tick()` to wait for pending DOM changes to resolve.

## Component Basics
- **Structure**: Familiar with the standard structure of a `.svelte` component.
- **HTML Injection**: Using the `{@html}` tag to render raw HTML strings.
- **Events**: Attaching handlers using lowercase attributes like `onclick`.
- **Lifecycles**: Using `onMount` and `onDestroy` for resource management.

## Directives & Binding
- **Value Binding**: Using `bind:value` for text/input state synchronization.
- **Check Binding**: Using `bind:checked` for boolean state (checkboxes).
- **DOM Binding**: Using `bind:this` to capture references to DOM elements.
